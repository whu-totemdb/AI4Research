## 研究轨迹

Cheng Long 的研究经历了从传统空间数据管理到前沿向量数据库系统的演进。早期研究聚焦于空间匹配问题、图数据管理等传统数据库领域，近年来全面转向高维向量搜索和向量数据库系统，建立了 VectorDB @NTU 研究组。2023-2025年间在 SIGMOD 连续发表多篇关于向量量化和图索引的重要工作，形成了从理论到系统的完整研究体系，其 RaBitQ 系列工作已成为该领域的标志性贡献。

## 花边与轶事

Cheng Long 在社交媒体上较为低调，主要通过学术渠道展示研究成果。其研究组 VectorDB @NTU 建立了完善的开源生态，RaBitQ Library 在 GitHub 上公开并持续维护。值得一提的是，他的博士生 Jianyang Gao 在2019年曾获得 ICPC 世界总决赛第62名，并在2025年获得 NTU CCDS 杰出博士论文奖（仅两位获奖者之一），体现了其培养学生的卓越能力。研究组近期在工业界影响力显著提升，受邀在 Ant Group、Zilliz、Microsoft、Meta、ByteDance、Huawei 等多家顶级科技公司进行技术分享。
