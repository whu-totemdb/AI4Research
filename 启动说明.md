# AI4Research 快速启动指南

## 一键启动

双击 **`start.bat`** 即可启动前后端服务：
- 后端会在独立窗口运行 (http://localhost:8000)
- 前端会在独立窗口运行 (http://localhost:3000)
- 启动完成后会自动打开浏览器

## 关闭服务

**方式 1**: 双击 **`stop.bat`** 关闭所有服务

**方式 2**: 直接关闭对应的命令行窗口
- 关闭 "AI4Research-Backend" 窗口停止后端
- 关闭 "AI4Research-Frontend" 窗口停止前端

## 手动启动（可选）

如果需要手动启动：

### 后端
```bash
cd backend
python main.py
```

### 前端
```bash
cd frontend
npm run dev
```

## 系统要求

- Python 3.8+
- Node.js 16+
- 已安装后端依赖 (`pip install -r requirements.txt`)
- 已安装前端依赖 (`npm install`)
